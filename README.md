# 🧠 Smart STEM Digital Multimeter

## 📌 Overview
The Smart STEM Digital Multimeter is a modular embedded system designed to measure electrical parameters while serving as a learning platform for electronics and instrumentation.

This project focuses on building a multimeter from scratch using a microcontroller, starting with a 0.91" OLED display and expanding toward advanced features like oscilloscope waveform visualization.

---

## 🎯 Objectives
- Develop a digital multimeter from scratch  
- Learn ADC, signal conditioning, and calibration  
- Implement auto-ranging techniques  
- Design an efficient embedded user interface  
- Expand into oscilloscope functionality  

---

## ✨ Features

### ✅ Current
- OLED Display (0.91", I2C)  
- DC Voltage Measurement  
- Resistance Measurement  
- Button-based Menu Navigation  

### 🚧 In Progress
- Auto-ranging system  
- Measurement calibration  
- Noise filtering  

### 🔮 Future
- TFT Display with waveform (oscilloscope)  
- Wireless data logging  
- Wearable design  
- Modular probe system  

---

## 🏗 System Architecture

### 🔧 Hardware
- Microcontroller (Raspberry Pi Pico)  
- Voltage Divider Circuit  
- Reference Resistor Network  
- Transistor-based switching  
- OLED Display (I2C)  
- Input probes  

### 💻 Software
- Arduino IDE (C/C++)  
- Modular structure:
  - display
  - voltage
  - resistance
  - menu
